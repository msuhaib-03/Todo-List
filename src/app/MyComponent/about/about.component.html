About This App

A modern Todo application built with Angular 17 using standalone components and the new control flow syntax.

What This App Does

Users can add, complete, and delete tasks with automatic persistence using localStorage. The UI updates instantly with Angular’s reactive rendering.

Tech Stack

  • Angular 17 (Standalone Components)
  • for & if Control Flow
  • TypeScript
  • Bootstrap 5
  • LocalStorage
